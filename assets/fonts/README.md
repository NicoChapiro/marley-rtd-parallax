# Fonts

Upload the official Marley Coffee font files manually into this folder.

Expected files:
- FestivoLettersNo1-Regular.otf
- GT-Pressura-Mono.otf
- GT-Pressura-Mono-Bold.otf

Do not commit font binaries through Codex unless explicitly approved.

These fonts are referenced by styles.css using relative paths.

If fonts do not load, check DevTools → Network → Font for 404s or filename mismatches. If fonts are missing, fallback fonts will be used.
