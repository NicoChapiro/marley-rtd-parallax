# Assets Ready to Drink

Sube manualmente en esta carpeta los assets reales de la landing Marley Coffee Ready to Drink.

No agregues binarios generados automáticamente por Codex. Los archivos de imagen deben cargarse manualmente con estos nombres exactos para que `index.html` y `script.js` los usen automáticamente:

- `rtd-cappuccino-250ml.png`
- `rtd-vainilla-latte-250ml.png`
- `rtd-caramel-250ml.png`
- `rtd-lifestyle-bg.jpg`
- `beans-cluster.png`
- `leaf-left.png`
- `leaf-right.png`
- `rasta-ribbon.png`
- `kraft-texture.jpg`
- `caramel-splash.png`
- `vanilla-flower.png`

Mientras estos archivos no existan, la landing mantiene fondos alternativos, gradientes y placeholders CSS para no romper la experiencia.
